# Gestor de Proyectos v2 · Motor de Tareas Diarias · Recouromex

Módulo conectado a Supabase (proyecto principal qcwpnovwyhzhqqmhjxco). Tareas
diarias por usuario (rutinas + puntuales), cierre de día perezoso, candado a lo
no hecho, reprogramación, tareas de Alta Dirección con prórroga, panel de
solicitudes y cierre mensual de proyectos. Prefijo de tablas `gpr2_` (no toca el
Gestor v1 `gpr_`).

Antes de usar: editar el bloque CONFIG en index.html con la URL y la anon key.
Los PIN viven en la tabla gpr2_usuarios (Rodrigo los inyecta; no van en el HTML).
